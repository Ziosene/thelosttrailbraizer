# Action Cards — The Lost Trailbraizer

> Target: ~300 carte | Completate: 300 | Rimanenti: 0 ✅

## Schema carta Azione

```
### [NUMERO]. Nome Carta
- **Tipo**: Offensiva / Difensiva / Economica / Manipolazione / Interferenza / Utilità
- **Quando**: In qualsiasi momento / Solo durante combattimento / Solo fuori combattimento / Fuori dal proprio turno
- **Effetto**: descrizione
- **Rarità**: Comune / Non comune / Raro / Leggendario
```

---

## Categorie

| Categoria | Descrizione | % del mazzo |
|---|---|---|
| **Economica** | Guadagna/ruba Licenze o Certificazioni | ~20% |
| **Offensiva** | Danno a boss o avversari | ~20% |
| **Difensiva** | Cura, protezioni, HP | ~15% |
| **Manipolazione dado** | Modifica risultati dado | ~15% |
| **Interferenza** | Carte giocabili durante il turno altrui | ~15% |
| **Utilità** | Pesca, scarto, gestione mano/mazzo | ~15% |
