# Boss Cards — The Lost Trailbraizer

> Target: 100 boss | Completati: 100 | ✅ COMPLETATO

## Schema carta Boss

```
### [NUMERO]. Nome Boss
- **HP**: X
- **Soglia dado**: X+ (su d10)
- **Abilità**: descrizione effetto speciale
- **Ricompensa**: X Licenze [+ Certificazione]
- **Difficoltà**: Facile / Media / Alta / Leggendaria
```

> I boss con **Certificazione** sono segnati con 🏆
> I boss **Leggendari** sono rari nel mazzo (1 copia invece di 2)
